## 使用前提

请务必仔细阅读本文档，否则可能会无法正常使用。

1. 确保分辨率为 1920x1080。（未来会支持更多分辨率）
2. 确保浏览器为 Chrome。
3. 确保浏览器缩放比例为 100%。
4. 确保浏览器隐藏了书签栏。
5. 确保浏览器保持全屏状态。
6. 确保已经在 Particle Network 网站上连接了钱包。
7. 确保已经在 Particle Network 钱包中添加了联系人，脚本会自动选择第一个联系人。
8. 确保已经在 Particle Network 钱包中具有足够的 ETH Seplolia 代币和足够的 USDG 代币。

## 使用方法

在命令行直接运行该文件即可。
